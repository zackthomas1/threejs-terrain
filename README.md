# threejs-terrain
A terrain generator created using three.js


## run dev environment
```
npx vite
```
```
npm run dev
```

## build
```
npx vite build
```

npx is installed with Node.js, and runs command line programs like Vite so that you don't have to search for the right file in node_modules/ yourself.

## run eslint

```
npm run lint
```
